# Account management¶

This section provides information on how various users can manage accounts
within DataRobot, including how administrators can manage organizations as
well as how individual users can manage their own DataRobot settings.

  * #### [Management toolbar](toolbar/index.html)¶

Describes how to use the navigation elements in the upper-right of DataRobot
to access walkthroughs, view help resources and documentation, check platform
notifications, use shortcuts for quick navigation, and access admin and
account settings.

  * #### [Administrator's guide](admin/index.html)¶

Provides information on how DataRobot administrators can manage their
organization within the platform, including SSO setup; tenant isolation and
collaboration; managing users and groups; configuring deployment approval
policies; and setting up network policy controls.

  * #### [Accounts settings](acct-settings/index.html)¶

Provides information on how DataRobot users can manage their accounts within
the platform, including 2FA, data connections, data connection credentials,
and API keys and tools.
